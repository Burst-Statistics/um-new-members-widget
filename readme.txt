== Description ==
Add-on to Ultimate member to show latest new members in a widget or with shortcode.
To use the widget, simply add the widget in Wordpress to your sidebar. To use the shortcode, add the shortcode umnm-new-members to a page.

Need any changes, other features? Post a support item, and if I like it, I'll add it.

Like this plugin? For more information see [really-simple-plugins.com/um-new-members](https://really-simple-plugins.com/um-new-members/).

For more Ultimate Member add-ons, see [really-simple-plugins.com/download/category/ultimate-member/](https://really-simple-plugins.com/download/category/ultimate-member/).

== Installation ==
To install this plugin:

1. Download the plugin
2. Upload the plugin to the wp-content/plugins directory,
3. Go to “plugins” in your WordPress admin, then click activate.
